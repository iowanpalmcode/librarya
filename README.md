# Digital Library for Learning Resources

This is an AI-powered digital library that curates useful links and websites for students and learners, presented visually as books.

## Features
- Curated learning resources organized by categories
- Visual book-like interface
- Search functionality
- AI-assisted categorization

## Installation
1. Install Python 3.8+
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run main.py`

## Usage
Run the app and browse the library of learning resources.
